
# Calculadora Web

## Ejecutar con Vite

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Iniciar la app:
   ```bash
   npm run dev
   ```

---

## Ejecutar pruebas con Deno

1. Instalar Deno:
   ```bash
   curl -fsSL https://deno.land/install.sh | sh
   ```

2. Ejecutar pruebas:
   ```bash
   deno test
   ```
